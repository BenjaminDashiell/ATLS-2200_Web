SuggestBot

Spring 2019 Data Structures Project

SuggestBot is a program that asks the user inputs keywords/topics they want to "search" for and our algorithm will run through the data stored in our binary search tree full of YouTube metadata and selected the best candidates for suggestions. The difference from YouTube's algorithm is once a user has watched a suggested video, it will not suggest it again, eliminating repetitive recommendations.
